# Gidex

Gidex is a tool for automatically synchronizing GitHub repositories as submodules in your local repository.

It allows you to fetch all repositories from specified organizations and add them as submodules, simplifying dependency management and collaboration.

## Installation

To install Gidex, you can clone the repository:

```sh
$ git clone https://github.com/masajinobe-ef/gidex.git
$ cd gidex
$ sudo chmod +x sync.sh
$ ./sync.sh
```
